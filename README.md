# 🤖 Chatbot Transformation Numérique

Chatbot RAG multilingue sur la transformation numérique.

## 🚀 Demo en ligne

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://chatbot-transformation-numerique-xxxx.streamlit.app)

## 🛠️ Stack technique

| Composant | Technologie |
|---|---|
| Embeddings | BAAI/bge-m3 (1024 dimensions) |
| Base vectorielle | Qdrant Cloud |
| LLM | Groq Llama 3.3 70B |
| Interface | Streamlit |
| Plateforme | Google Colab |
